# 🎬 Movie Recommendation System

A Machine Learning powered Movie Recommendation System built using **Python**, **Streamlit**, and **Scikit-learn**. This application recommends movies similar to the one selected by the user using a content-based recommendation algorithm.

---

## ✨ Features

* 🎥 Recommend similar movies instantly
* 🖼️ Fetch movie posters using the TMDB API
* ⚡ Fast content-based recommendation engine
* 🎨 Clean and responsive Streamlit interface
* 📚 Machine Learning powered recommendation system

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Pickle
* TMDB API

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Zain-Khan-AI/Movies-Recommendation-System.git
```

Go to the project folder:

```bash
cd Movies-Recommendation-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run main.py
```
